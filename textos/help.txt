*Sessão Help*
*Comandos Disponíveis:*

1. `!teste`: Envia uma mensagem de teste para o usuário que o enviou.

   Exemplo: `!teste`

2. `!apresentar`: Envia uma mensagem de apresentação do RPG.

   Exemplo: `!apresentar`

3. `!enquete`: Envia uma enquete de teste para o usuário que a enviou.

   Exemplo: `!enquete`

4. `!sendDM`: Envia uma mensagem direta para um número específico.

   Sintaxe: `!sendDM numero texto`

   Exemplo: `!sendDM 551888003813 Olá, isso é uma mensagem direta!`

**Observações:**
- Certifique-se de seguir a sintaxe correta para o comando `!sendDM`.
- O número no comando `!sendDM` deve estar no formato que contém apenas o número do contato, por exemplo, `551888003813`.
- Este chatbot está em desenvolvimento, e novas funcionalidades podem ser adicionadas no futuro.

Para obter ajuda sobre um comando específico, digite `!help` seguido de comando. Por exemplo, `!help !teste`.
